# MetricMind

## 📌 Project Overview

MetricMind is an AI-powered Business Intelligence (BI) platform that allows users to ask business questions in natural language and receive trusted, data-driven insights.

Instead of generating raw SQL, the system interacts with a Semantic Layer to ensure consistent and governed business metrics.

## 🚀 Tech Stack

- Next.js
- LangChain
- Cube.dev
- dbt
- Snowflake
- Git & GitHub
